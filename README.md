Netflix Content Trends Dashboard

 Project Overview
This project analyzes Netflix’s content library to uncover trends in content type, production countries, genre distribution, and content growth over time. The analysis is visualized using an interactive dashboard built in Tableau.

The goal is to transform raw Netflix data into meaningful insights that help understand how the platform’s content catalog has evolved.

  Data Cleaning & Preparation

The dataset was cleaned and structured in Excel before visualization:

Standardized text fields (country, genre)
Handled missing values (e.g., “Unknown” categories)
Converted date_added into proper date format
Split and normalized duration fields where needed
Removed inconsistencies and ensured uniform formatting

The cleaned dataset was then imported into Tableau for analysis.

  Key Visualizations
1. Content Type Distribution
 Shows the breakdown between Movies and TV Shows available on Netflix.
2. Content Growth Over Time
 Displays how Netflix’s content library has expanded over the years, highlighting significant growth after 2015.
3. Top Content-Producing Countries
 Identifies countries contributing the most content, with the United States leading by a large margin.
4. Most Common Genres
 Highlights the most frequent content categories, with dramas and international movies being the most prevalent.

  Key Insights
- Movies make up the majority of Netflix’s catalog compared to TV shows
- Netflix experienced rapid content expansion after 2015
- The United States is the largest contributor of Netflix content
- Drama and international genres dominate the platform’s library
